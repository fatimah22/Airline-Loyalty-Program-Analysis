# Airline Loyalty Program Analysis

## Overview
This project presents an interactive **Power BI** analysis of an airline loyalty program, designed to evaluate customer acquisition patterns, segment behavior, and the real impact of a 2018 promotional campaign.

The main objective was not only to measure growth in loyalty memberships, but also to determine whether that growth was truly incremental or partly driven by demand cannibalization.

## Problem
The airline loyalty program lacked a clear analytical view of acquisition behavior and campaign effectiveness.

Although membership growth was visible during the 2018 promotional campaign, it was unclear whether the increase reflected genuinely new customers or simply a shift in demand that would have happened anyway.

## Solution
An interactive **Power BI dashboard** was built to analyze:

- Historical customer acquisition trends
- Segment-level performance
- Customer behavior patterns
- Campaign-driven versus organic membership growth

The report was designed as a storytelling dashboard to separate promotional lift from underlying baseline demand.

## Key Insight
The analysis quantified a **21% acquisition uplift** during the campaign period.

However, the results also suggested that only around **600 customers were truly net-new**, while approximately **38% of the uplift represented cannibalized demand** rather than fully incremental growth.

## Dashboard Focus Areas
The report includes analysis across the following areas:

1. **Acquisition Trends**  
   Tracks how loyalty membership changed over time and highlights campaign-period spikes.

2. **Segment Performance**  
   Compares performance across customer groups to understand which segments contributed most to the observed growth.

3. **Campaign Uplift View**  
   Separates incremental campaign effect from organic or shifted demand.

4. **Customer Behavior**  
   Examines how customers responded during and around the promotional period.

## Business Value
This project provides a clearer view of marketing effectiveness by distinguishing between apparent growth and truly incremental customer acquisition.

Instead of treating all campaign-period growth as success, the analysis supports more accurate evaluation of promotional performance and helps guide better future campaign decisions.
